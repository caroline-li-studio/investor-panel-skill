# Social Promotion

Use this when writing launch posts or friendly comments for X/Twitter, 小红书, 即刻, or similar communities.

## Launch Post Shape

Use a casual, specific tone:

```text
做了一个 <project>。

我的痛点是：<real pain>.

所以我做了 <project>：
<one vivid explanation>.

它会输出：
- <result 1>
- <result 2>
- <result 3>

Repo:
<url>
```

## Short Version

```text
做了个 <project>。

不是 <wrong mental model>，而是 <better mental model>。

适合用来 <use case 1> / <use case 2> / <use case 3>。

<url>
```

## Friendly Comment Style

Do not spam links. Reply only when the original post is relevant.

Good comment:

```text
这个方向我最近也在试一个小工具：<one-sentence project value>。

不是硬广，刚好和你这条相关，贴一下：
<url>
```

Better comment:

1. First respond to the person's actual point.
2. Then mention the repo as a related experiment.
3. Keep it short.

## Avoid

- "重磅发布"
- "全网首个"
- "颠覆"
- "求 star"
- Copy-pasting the same comment everywhere.
- Promising results the project cannot guarantee.
