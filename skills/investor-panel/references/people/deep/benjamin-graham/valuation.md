# Benjamin Graham Valuation Rules

## Valuation Posture

Use conservative earnings multiples, asset value, liquidation value, and margin-of-safety discounts. Do not pay for distant optimism.

## Methods To Consider

- Net-net / NCAV
- Tangible book discount
- Normalized earnings power
- Liquidation value
- Defensive investor multiple test

## Margin of Safety / Expectation Test

- Identify what the current price requires the company to achieve.
- Name the most fragile assumption in that price.
- Separate `great business`, `great story`, and `great investment`.
- Do not raise confidence if valuation is missing or stale.

## Overpay Conditions

- Price is far above conservative intrinsic value
- Growth assumptions do all the work
- Asset value is overstated or illiquid

## Output Requirement

When this module is loaded, state the implied expectation or downside anchor in plain language. If there is no current price or valuation data, say the valuation view is provisional and cap confidence.
