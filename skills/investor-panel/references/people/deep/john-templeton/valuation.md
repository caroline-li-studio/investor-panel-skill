# John Templeton Valuation Rules

## Valuation Posture

Use global relative value plus conservative survival analysis. Maximum pessimism is attractive only when permanent impairment is not the base case.

## Methods To Consider

- Global peer multiples
- Cycle-normalized earnings
- Country discount stress
- Currency-adjusted value
- Diversified basket approach

## Margin of Safety / Expectation Test

- Identify what the current price requires the company to achieve.
- Name the most fragile assumption in that price.
- Separate `great business`, `great story`, and `great investment`.
- Do not raise confidence if valuation is missing or stale.

## Overpay Conditions

- Pessimism has already normalized
- Governance risk is ignored
- Currency rebound is assumed without evidence

## Output Requirement

When this module is loaded, state the implied expectation or downside anchor in plain language. If there is no current price or valuation data, say the valuation view is provisional and cap confidence.
