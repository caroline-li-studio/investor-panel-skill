# David Swensen Valuation Rules

## Valuation Posture

Value the asset in portfolio context: expected return net of fees, liquidity premium, governance risk, and correlation benefit.

## Methods To Consider

- Expected return decomposition
- Illiquidity premium test
- Peer public-market equivalent
- Scenario stress for liquidity
- Net-of-fee return estimate

## Margin of Safety / Expectation Test

- Identify what the current price requires the company to achieve.
- Name the most fragile assumption in that price.
- Separate `great business`, `great story`, and `great investment`.
- Do not raise confidence if valuation is missing or stale.

## Overpay Conditions

- Illiquidity premium is absent
- Fees consume alpha
- Asset duplicates existing exposure
- Recent returns anchor expectations

## Output Requirement

When this module is loaded, state the implied expectation or downside anchor in plain language. If there is no current price or valuation data, say the valuation view is provisional and cap confidence.
