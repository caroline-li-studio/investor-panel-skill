# Julian Robertson Valuation Rules

## Valuation Posture

Value on a relative basis: pay for superior winner economics only if spread versus peers is justified and not crowded.

## Methods To Consider

- Peer multiple spread
- Long/short pair expected return
- Revision-adjusted valuation
- SOTP for relative winners
- Short downside scenario

## Margin of Safety / Expectation Test

- Identify what the current price requires the company to achieve.
- Name the most fragile assumption in that price.
- Separate `great business`, `great story`, and `great investment`.
- Do not raise confidence if valuation is missing or stale.

## Overpay Conditions

- Winner status is consensus and crowded
- Valuation spread assumes flawless execution
- Short side lacks catalyst or borrow

## Output Requirement

When this module is loaded, state the implied expectation or downside anchor in plain language. If there is no current price or valuation data, say the valuation view is provisional and cap confidence.
