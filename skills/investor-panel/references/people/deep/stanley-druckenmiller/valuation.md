# Stanley Druckenmiller Valuation Rules

## Valuation Posture

Valuation is judged against cycle position and asymmetry. A high multiple can work with accelerating liquidity; a low multiple can fail in tightening.

## Methods To Consider

- Asymmetric payoff map
- Revision-adjusted multiple
- Macro scenario sensitivity
- Trend/risk stop map
- Cycle-normalized value

## Margin of Safety / Expectation Test

- Identify what the current price requires the company to achieve.
- Name the most fragile assumption in that price.
- Separate `great business`, `great story`, and `great investment`.
- Do not raise confidence if valuation is missing or stale.

## Overpay Conditions

- Trade is crowded and upside/downside no longer asymmetric
- Macro tailwind fully priced
- Stop is too far from thesis value

## Output Requirement

When this module is loaded, state the implied expectation or downside anchor in plain language. If there is no current price or valuation data, say the valuation view is provisional and cap confidence.
