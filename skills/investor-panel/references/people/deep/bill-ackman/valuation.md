# Bill Ackman Valuation Rules

## Valuation Posture

Use sum-of-parts, normalized earnings, strategic value, and catalyst-adjusted scenarios. The investment must work on economics, not publicity.

## Methods To Consider

- SOTP
- Normalized EBITDA/FCF
- Before/after catalyst value
- Private market value
- Downside value if activism fails

## Margin of Safety / Expectation Test

- Identify what the current price requires the company to achieve.
- Name the most fragile assumption in that price.
- Separate `great business`, `great story`, and `great investment`.
- Do not raise confidence if valuation is missing or stale.

## Overpay Conditions

- Catalyst upside is already priced
- Governance change is assumed but not probable
- Public presentation selects aggressive peers

## Output Requirement

When this module is loaded, state the implied expectation or downside anchor in plain language. If there is no current price or valuation data, say the valuation view is provisional and cap confidence.
